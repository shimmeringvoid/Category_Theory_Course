The course

Each lesson is a self-contained page: motivation, a boxed definition, worked examples with commutative diagrams that draw themselves, a §5 where the day’s concept is implemented and machine-checked in Python (every output shown was actually produced by the code), and a set of exercises with hidden solutions — from warm-ups to arrows-only proofs. The lessons build strictly in order; start at 01.

Yoneda & Monads

The finale. The Yoneda lemma turns the course’s slogan into a theorem — an object is exactly the arrows into it, proved by tracking the identity and counted by machine — and every adjunction folds into a monad: List, Maybe, and Writer, with the Kleisli category revealing the monad laws as Lesson 1’s two axioms in disguise.

How to use these pages

Keep all the .html files in one folder so the links between lessons work. Each lesson ships with a matching .py file containing the lesson’s verified code, runnable with plain Python 3 — no packages required. Do the exercises before opening the solutions; the arrows-only ones are the course. Each lesson also ships as a JupyterLab notebook (in notebooks/) containing all of its verified code, ready to run and modify.
